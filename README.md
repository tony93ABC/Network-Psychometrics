# Network Psychometrics 🧠🕸️

Welcome to my repository dedicated to the study and application of Network Psychometrics! 

## 📖 About this Repository

This repository serves as my personal learning hub and code archive for network analysis in psychology and the social sciences. The theoretical explanations, core concepts, and main scripts found here are heavily based on the excellent book:

> **Network Psychometrics with R: A Guide for Behavioral and Social Scientists** > *by Adela-Maria Isvoranu, Sacha Epskamp, Lourens J. Waldorp, and Denny Borsboom.*

If you are interested in this field, I highly recommend checking out their work!

## 🚀 What's Inside?

While the backbone of this repository follows the structure and teachings of the book, I have also included my own **personal additions and experiments**, such as:

* **Personal Notes & Summaries:** Simplified explanations of complex network concepts.
* **Practice Exercises:** Custom R scripts where I apply network modeling techniques.
* **Real-World Applications:** Hands-on exercises analyzing various **public datasets** to see how these methods perform in practical scenarios outside the textbook.

## 🛠️ Tools & Libraries

The code in this repository is written in **R**. If you want to run the scripts, you will likely need the following primary packages:
* `bootnet`
* `qgraph`
* `networktools`
* `IsingFit`
* `mgm`

*(Note: each script or folder may have its own specific dependencies).*
